CSS Specificity Explained

  • Specificity determines which CSS rule is applied when multiple rules match the same element.
  • It is calculated based on the types of selectors used in the rule.
  • Inline styles have the highest specificity (e.g., style="color: red;").
  • IDs have higher specificity than classes, attributes, and pseudo-classes.
  • Classes, attributes, and pseudo-classes have higher specificity than element selectors and pseudo-elements.
  • When specificity is the same, the last defined rule in the CSS is applied.

Examples:

  • #id { color: blue; } has higher specificity than .class { color: red; }.
  • .class { color: red; } or Attribute:   a[href*="example"] { font-size: 2em; } or Pseudo:   a:hover { background-color: yellow; } has higher specificity than div { color: green; }.
  • Inline styles, e.g., style="color: yellow;", override all external or internal CSS rules.
  • !important override all CSS rules including Inline Styles.
  • Inline style > ID selector > Class or attribute selector > Element selector > Universal selector.

Values according to specificity:

Selector Value
Universal 0
Element 1
Class or Attribute or Pseudo selector 10
ID 100
Inline 1000
!important 10000
