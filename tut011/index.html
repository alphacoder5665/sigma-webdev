What are Semantic Tags?

Semantic tags are HTML tags that convey meaning about the content they contain. They help search engines and browsers understand the structure and purpose of the content on a webpage.

For example, the <header> tag indicates the header section of a webpage, while the <footer> tag indicates the footer section. Other examples of semantic tags include <article>, <section>, and <nav>.

Using semantic tags can improve the accessibility of your website, as screen readers and other assistive technologies can better understand the structure of your content. Additionally, semantic tags can help with search engine optimization (SEO) by providing context to search engines about the content on your page.

Here are some common semantic tags and their purposes:

  • <header>: Represents the header section of a webpage or a section.
  • <nav>: Represents a navigation section.
  • <article>: Represents a self-contained piece of content.
  • <section>: Represents a thematic grouping of content.
  • <aside>: Represents content that is tangentially related to the main content.
  • <footer>: Represents the footer section of a webpage or a section.
  • <main>: Represents the main content of a document.
  • <figure>: Represents self-contained content, such as an image or diagram.
  • <figcaption>: Represents a caption for a <figure>.
  • <time>: Represents a specific time or date.
  • <mark>: Represents highlighted or marked text.
  • <address>: Represents contact information for the author or owner of a document or article.
  • <details>: Represents a disclosure widget from which the user can obtain additional information or controls.
  • <summary>: Represents a summary, heading, or legend for a <details> element.
